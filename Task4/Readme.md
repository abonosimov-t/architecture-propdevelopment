# Задание 4. Защита доступа к кластеру Kubernetes


##  Таблица ролей в Kubernetes
| Роль | Права роли | Группы пользователей |
| --- | --- | --- |
| cluster-admin | Полный доступ ко всем ресурсам | Администраторы кластера |
| security-admin | Доступ к секретам, аудит, просмотр логов | Специалисты ИБ |
| devops-engineer | Управление деплоями, сервисами, конфигурациями | DevOps инженеры |
| developer | Просмотр логов, деплоев, сервисов | Разработчики |
| viewer | Только просмотр ресурсов | Аналитики |
